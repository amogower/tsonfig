# tsconfig

> Standard [TypeScript config](https://www.typescriptlang.org/docs/handbook/tsconfig-json.html)

## Install

```sh
npm install --save-dev @amogower/tsconfig
```

## Usage

`tsconfig.json`

```json
{
  "extends": "@amogower/tsconfig",
  "compilerOptions": {
    "outDir": "dist",
    "lib": [
      "es2018"
    ]
  }
}
```
